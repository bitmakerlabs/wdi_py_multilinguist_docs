Welcome to Multilinguist’s documentation!¶

class multilinguist.Multilinguist¶

This class represents a world traveller who knows what languages are spoken in each country around the world and can cobble together a sentence in most of them (but not very well)

language_in(country_name)¶

Uses the RestCountries API to look up one of the languages spoken in a given country

Parameters:country_name (str) – The full name of a country.
Returns:2 letter iso639_1 language code.
Return type:bool

Returns:A rough translation of msg.
